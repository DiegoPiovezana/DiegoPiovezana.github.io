Instruções:

Para girar a pirâmide:
W = Cima;
S = Baixo;
A = Direita;
D = Esquerda.
*Para girar o cubo use as setas direcionais.

Para controlar zoom :
Q = Afasta câmera;
E = Aproxima câmera.

Para deslocar cubo :
PageUp = Afasta;
PageDown = Aproxima.

Para cubo orbitar em função da rotação da pirâmide:
R = Ativa;
T = Desativa.

Para cubo orbitar de maneira independente:
L = Acelera órbita no sentido anti-horário;
Ç = Acelera órbita no sentido horário.
P = Interrompe a órbita independete do cubo.
